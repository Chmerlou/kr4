# Персональный трекер освоения технологий

Одностраничное приложение (SPA) на React для систематизации и отслеживания прогресса в изучении технологий по дорожной карте.

## Возможности

- ✅ Загрузка дорожной карты из JSON-файла
- ✅ Экспорт дорожной карты с пользовательскими данными
- ✅ Визуальное представление карты в виде сетки карточек
- ✅ Детальный просмотр каждого пункта
- ✅ Система статусов: Не начат, В работе, Выполнено
- ✅ Добавление и редактирование заметок к пунктам
- ✅ Планирование сроков завершения
- ✅ Отображение общего прогресса в процентах
- ✅ Сохранение данных в localStorage

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для production
npm run build
```

## Формат JSON-файла дорожной карты

```json
{
  "title": "Название дорожной карты",
  "description": "Описание дорожной карты (опционально)",
  "items": [
    {
      "id": "уникальный-идентификатор",
      "title": "Название пункта",
      "description": "Описание пункта (опционально)",
      "links": [
        {
          "title": "Название ссылки",
          "url": "https://example.com"
        }
      ]
    }
  ]
}
```

## Пример использования

1. Запустите приложение командой `npm run dev`
2. Нажмите кнопку "Загрузить карту" в шапке
3. Выберите JSON-файл с дорожной картой (пример: `public/example-roadmap.json`)
4. Просматривайте карточки на главной странице
5. Кликайте на карточку для просмотра деталей
6. Управляйте статусами, добавляйте заметки и устанавливайте сроки
7. Экспортируйте обновленную дорожную карту с вашими данными

## Структура проекта

```

├── components/
│   ├── Header.jsx          # Шапка с загрузкой/экспортом и прогресс-баром

│   ├── RoadmapCard.jsx     # Карточка отдельного пункта
│   └── RoadmapDetail.jsx   # Детальная страница пункта
├── App.jsx                 # Главный компонент с роутингом
└── main.jsx                # Точка входа
```
